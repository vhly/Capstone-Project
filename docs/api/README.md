# D3 API


## Icon API

http://media.blizzard.com/d3/icons/\<type\>/\<size\>/\<icon\>.png

http://media.blizzard.com/d3/icons/portraits/42/demonhunter_female.png

### type parameter

```
   "items" or "skills"

```

### size parameter

```
    small
    large
```

### icon parameter

```
unique_helm_set_07_x1_demonhunter_male

http://media.blizzard.com/d3/icons/<type>/<size>/<icon>.png

```
